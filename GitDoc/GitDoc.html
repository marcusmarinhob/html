  • Comandos do Git

    • git help <command>: Abre a documentação do comando digitado no espaço definido por <command>.
    • git init RepositoryName: Cria um novo repositório denominado RepositoryName.
    • git add FileName: Prepara as mudanças adicionadas no arquivo FileName para serem adicionadas ao repositório local.
    • git commit: Adiciona a mudança ao repositório local.
    • git status: Ver o status do repositório.
    • git log: Ver os commits realizados no repositório.
